## 用户需求

在 PC Web 端实现 PPT 文件的在线预览功能。

## 核心功能

- 在 Assembly（PPT 组装）页面添加完整的 PPT 预览功能
- 支持查看完整的 PPT 内容（不仅仅是缩略图）
- 支持翻页、缩放等基本操作
- 预览效果接近原始 PPT

## 技术选型

选择**腾讯云数据万象（CI）文档预览服务**作为解决方案，理由：

- 项目已使用 CloudBase，可无缝对接腾讯云生态
- 接入简单，只需修改 URL 参数
- 成本可控（按量计费，免费额度充足）
- 支持完整 PPT 格式和动画效果

## 技术栈

- **前端**：React 18 + TypeScript + TDesign UI
- **后端**：Python FastAPI + 腾讯云 COS SDK
- **云服务**：腾讯云数据万象（CI）文档预览

## 实现方案

### 方案架构

```
用户点击预览 → 前端请求预览 URL → 后端生成 PPT 并上传到 COS → 返回预览 URL → 前端 iframe 加载预览页面 → 腾讯云 CI 实时转换展示
```

### 预览 URL 格式

```
{原始文件URL}?ci-process=doc-preview&dstType=html
```

### 核心技术点

#### 1. 前端预览组件

使用 iframe 嵌入腾讯云预览页面，支持多种预览模式：

- HTML 模式：交互性好，支持复制
- 图片模式：保真度高，支持缩放

#### 2. 后端 COS 上传

- 使用腾讯云 COS Python SDK 上传文件
- 生成带签名的临时访问 URL
- 支持文件过期自动清理

#### 3. 数据万象服务配置

- 在 COS 控制台开通文档预览功能
- 配置预览参数和样式

## 性能优化

- 小文件（<5MB）：预览时间 < 3秒
- 中等文件（5-20MB）：预览时间 < 10秒
- 大文件（>20MB）：显示加载进度提示

## 成本估算

- 文档转 HTML：0.01 元/次
- 文档转图片：0.1 元/千页
- 免费额度：6000 页/2 个月
- 预估成本（1000 次预览/月）：约 10-20 元

## 设计风格

采用与现有项目一致的 TDesign 设计风格，保持界面简洁专业。

## 预览组件设计

- **对话框预览**：使用 TDesign Dialog 组件，支持全屏切换

- **错误处理**：使用 MessagePlugin 显示错误提示
- **工具栏**：包含翻页、缩放、全屏等操作按钮

- 支持鼠标滚轮缩放
- 提供下载按钮作为备选方案
